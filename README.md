## Preview

```txt
$$$  6  5  4  3  2  1  HOME  1  2  3  4  5  6  $$$

Players roll dice and push the character toward their side of the board.
First player to reach the end wins.

---

Features

- Terminal-based gameplay
- ANSI colored UI
- Dice animations
- Coin toss system
- Two-player mode
- Pure standard C++
- No external libraries

---

Build & Run

g++ -std=c++17 house_game.cpp -o house_game
./house_game
